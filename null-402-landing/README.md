# null-402-landing

Marketing landing page for **null-402** — private pay-per-call on Stellar.

A single static `index.html`, no build step. Open it directly or deploy to any
static host (Vercel, Netlify, GitHub Pages, Cloudflare Pages).

```bash
# preview locally
python3 -m http.server 4321   # → http://localhost:4321
```

See [`OVERVIEW.md`](OVERVIEW.md) for the full project overview, and the sibling
repos: `null-402-sdk`, `null-402-gateway`, `null-402-dashboard`,
`null-402-contracts`, `null-402-circuits`.

> Static for now — swap to Next.js later if it needs CMS/interactivity.
