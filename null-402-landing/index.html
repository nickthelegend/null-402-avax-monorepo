API payments.
Nothing leaks.

x402 broadcasts who paid, how much, and which API they hit. null-402 replaces the payment with a zero-knowledge proof verified on Stellar — only a one-time nullifier and a valid: true ever appear.

Standard x402 (public)

Agent → 402 → token transfer (on-chain) → API Sender, amount, and endpoint are permanently indexable by anyone.

null-402 (private)

Agent → 402 → Groth16 proof on Stellar → API Proof generated client-side; secrets never leave the device.

What stays private

A shielded pool + on-chain Groth16 verification — real privacy, not a public transfer with a label.
